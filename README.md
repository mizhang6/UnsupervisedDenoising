# UnsupervisedDenoising
SAE_pytorch.py is a simple and easy-to-use unsupervised seismic random denoising code. In denoising research, we constantly strive for minimal signal leakage.

The codes correspond to the following paper:
[1] M. Zhang, Y. Liu, M. Bai and Y. Chen, "Seismic Noise Attenuation Using Unsupervised Sparse Feature Learning," in IEEE Transactions on Geoscience and Remote Sensing, vol. 57, no. 12, pp. 9709-9723, Dec. 2019, doi: 10.1109/TGRS.2019.2928715. 
[2] M. Zhang, Y. Liu and Y. Chen, "Unsupervised Seismic Random Noise Attenuation Based on Deep Convolutional Neural Network," in IEEE Access, vol. 7, pp. 179810-179822, 2019, doi: 10.1109/ACCESS.2019.2959238.
